# @live-software/ui-kit

Shared UI foundation for generic, reusable app primitives.

## Owns

- Generic UI wrappers.
- App-level UI primitives when they are shared by more than one app.
- Theme exports when theme work is introduced.
- Shared empty, error, and loading state components when they are generic.

## Does not own

- Training-specific components.
- Finance-specific components.
- Backend logic.
- Business or domain logic.

## Current state

This package is a skeleton only. It exposes no runtime UI API yet.
