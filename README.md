# Software Vida Personal

Proyecto personal full-stack y laboratorio de aprendizaje para arquitectura frontend/backend moderna.

## Objetivo

Construir una webapp responsive para gestionar vida personal, empezando por:

- Dashboard
- Entrenamiento
- Backend real
- Autenticación con Keycloak

Y usar el proyecto para aprender:

- React 19
- Microfrontends
- Module Federation
- Backend modular
- TypeScript estricto
- Accesibilidad
- CSS Modules
- Mantine
- Zustand
- TanStack Query
- Testing
- Performance
- Docker
- GitHub Projects
- Codex + agentes
- Feature flags
- Analytics
- Observabilidad
- SEO/rendering
- Service Workers
- Event bus

## Cómo usar este seed

1. Crea un repositorio vacío llamado `live-software`.
2. Descomprime este zip en la raíz del repo.
3. Lee `AGENTS.md`, `docs/product/vision.md` y `docs/architecture/system-overview.md`.
4. Abre Codex en VS Code.
5. Usa el prompt de `docs/playbooks/start-with-codex.md`.
6. No generes código grande antes de aprobar el plan de Fase 1.

## Estado

Este paquete no contiene todavía la app. Contiene la documentación, reglas, ADRs, skills y playbooks para que Codex arranque el proyecto con dirección clara.
