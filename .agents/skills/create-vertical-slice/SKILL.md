# Create Vertical Slice

Workflow:

1. Read issue.
2. Read relevant docs.
3. Identify affected app/package/backend module.
4. Define acceptance criteria.
5. Implement smallest useful slice.
6. Add loading/error/empty/forbidden states.
7. Add tests by risk.
8. Run validation.
9. Summarize changes.
