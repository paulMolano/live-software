# Architecture Review

Check:

- product value;
- learning value;
- boundaries;
- contracts;
- complexity;
- testability;
- maintainability;
- DX;
- performance;
- security.

Output:

1. Verdict.
2. Good signs.
3. Risks.
4. Missing decisions.
5. Recommended next step.
