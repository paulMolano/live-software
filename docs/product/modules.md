# Módulos del producto

## Inicial

### Dashboard

Vista global.

Contendrá inicialmente:

- resumen de entrenamiento;
- cards configurables;
- estado de sesión;
- placeholders de módulos futuros.

### Entrenamiento

Primer dominio real.

Contendrá:

- catálogo de ejercicios;
- detalle de ejercicio;
- crear/editar ejercicio;
- músculos;
- equipamiento;
- tips;
- media futura;
- rutinas futuras;
- progreso futuro.

## Futuro

### Finanzas

- gastos;
- ingresos;
- deudas;
- presupuesto;
- ahorro;
- simulaciones.

### Hábitos

- sueño;
- lectura;
- higiene;
- foco;
- check-ins diarios.

### Aprendizaje

- cursos;
- libros;
- notas;
- roadmap técnico;
- conocimiento generado por el proyecto.

### Familia / agenda

- custodia;
- planes;
- tareas;
- eventos.

### Knowledge base

- documentación técnica;
- aprendizajes;
- auditoría futura de vigencia.
