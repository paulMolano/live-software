# Visión de producto

## Producto

`Software Vida Personal` será una webapp personal para organizar entrenamiento, hábitos, finanzas, aprendizaje, familia, agenda y conocimiento técnico.

## Objetivo inicial

Construir primero:

- Dashboard
- Entrenamiento
- Backend real
- Autenticación preparada con Keycloak

## Objetivo secundario

Usar el proyecto como laboratorio de aprendizaje para:

- React 19
- Microfrontends
- Module Federation
- Backend modular
- Keycloak
- TypeScript estricto
- UI library profesional
- Accesibilidad
- Testing
- Performance
- Docker
- GitHub Projects
- Codex y agentes
- Analytics
- Feature flags
- Observabilidad
- SEO/rendering
- Service Workers
- Event bus

## Regla de producto

No construir todos los módulos a la vez.

Primero una vertical real:

`Dashboard + Training + API`
