# Start with Codex

Pega esto en Codex en VS Code:

```txt
Lee AGENTS.md, PLANS.md, docs/product/vision.md, docs/product/learning-goals.md y docs/architecture/system-overview.md.

Quiero empezar la Fase 1: crear el monorepo base.

Antes de implementar:
1. Explora el repo.
2. Resume qué existe.
3. Propón plan.
4. Lista comandos.
5. Lista archivos a crear.
6. Detecta riesgos.
7. No implementes hasta que el plan esté claro.

Restricciones:
- no crear features todavía;
- no crear más de shell, training-mfe y api;
- no añadir dependencias sin justificar;
- no saltarse TypeScript strict;
- no tocar Next.js todavía.
```
